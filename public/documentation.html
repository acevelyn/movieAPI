METHOD URL REQUEST RESPONSE
GET /movies N/A A JSON Object with data for all movies
GET /movies/[Title] N/A A JSON Object with data on a specific Movie
GET /genre/[Name] N/A A JSON Object with data on a specific Genre
GET /director/[Name] N/A A JSON Object with data on a certain Director
GET /users N/A A JSON Object with data for all Users
GET /users/[Username] N/A A JSON Object with data on a specific User
PUT /users/[Username] JSON Object User Data
{"Password": "test123"} 		A JSON Object with updated data for that User

POST /users/[Username]/movies/[MovieID] N/A A Message stating the movie has been added to favorites
DELETE /users/[Username]/movies/[MovieID] N/A A Message stating the movie has been removed from favorites
DELETE /users/[Username] N/A A Message stating the User has been removed.
