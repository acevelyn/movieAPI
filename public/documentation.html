Documentation Page

Request URL Response
GET /movies Returns list of all movies to user
GET /movies/[Title] Returns data about a single movie to the user
GET /movies/[Title]/genre Return data about genre by name/title
GET /movies/directors/[Name] Return data about a director by name
POST /users/ Allow new users to register
PUT /users/[Username] Allow users to update their user info
POST /movies Allow users to add a movie to their favorites list
DELETE /movies/[ID] Allow users to remove a movie from their favorites list
DELETE /users/ Allow users to deregister
